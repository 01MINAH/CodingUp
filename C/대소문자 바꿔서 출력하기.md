### *대소문자 바꿔서 출력하기*
*문제*
영어 알파벳으로 이루어진 문자열 str이 주어집니다. 각 알파벳을 대문자는 소문자로 소문자는 대문자로 변환해서 출력하는 코드를 작성해 보세요.
```C
#include <stdio.h>

void toggleCase(char str[]) {
    for (int i = 0; str[i] != '\0'; i++) {
        if (str[i] >= 'a' && str[i] <= 'z') {
            // 소문자를 대문자로 변환
            str[i] = str[i] - ('a' - 'A');
        } else if (str[i] >= 'A' && str[i] <= 'Z') {
            // 대문자를 소문자로 변환
            str[i] = str[i] + ('a' - 'A');
        }
    }
}

int main() {
    char inputString[100]={"aBcDeFg"};


    // 대문자와 소문자를 변환하여 출력
    toggleCase(inputString);
    printf(inputString);

    return 0;
}
```
